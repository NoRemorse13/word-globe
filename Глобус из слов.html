🔍 Возможные причины

  1. Файл называется word_globe.html, а GitHub Pages по умолчанию ищет файл с именем index.html. Если такого файла нет, сайт может не отображаться.

  2. Отсутствует файл .nojekyll. GitHub Pages по умолчанию использует Jekyll для обработки сайтов. Если ты не используешь Jekyll, наличие этого файла в корне репозитория отключит его.

  • Нажми на значок карандаша (редактировать).

  • Измени имя файла на index.html.
